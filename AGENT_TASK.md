# Agent Task: Set up Next.js Frontend Project

## Task Overview
Set up a Next.js 14 frontend project with App Router, TypeScript, Tailwind CSS, and shadcn/ui for the decomposition pipeline control center.

## Requirements (from brainstorm_1.md)
- Next.js 14 with App Router
- TypeScript for type safety
- Tailwind CSS for styling
- shadcn/ui components (Radix primitives)
- Real-time updates capability (SSE/WebSocket ready)
- Visualization libraries (React Flow, D3.js, Recharts)
- Markdown rendering support (react-markdown with remark-gfm)
- State management (Zustand or React Context)

## Objectives
1. Initialize Next.js project in frontend/ directory with App Router and TypeScript
2. Configure Tailwind CSS
3. Install and configure shadcn/ui
4. Set up project structure for the control center UI
5. Create basic layout and page structure
6. Install visualization and utility libraries
7. Write basic tests
8. Verify all tests pass

## Success Criteria
- Next.js project initialized with correct configuration
- Tailwind CSS working properly
- shadcn/ui components configured and importable
- Basic layout structure in place
- All dependencies installed
- Tests passing
- Ready for control center implementation

## Technical Details
- Framework: Next.js 14+ with App Router
- Language: TypeScript
- Styling: Tailwind CSS
- Component Library: shadcn/ui (Radix primitives)
- Testing: Jest + React Testing Library (or Vitest)

## Implementation Plan
1. Create frontend/ directory
2. Initialize Next.js with TypeScript and Tailwind
3. Configure Tailwind CSS
4. Initialize shadcn/ui
5. Install additional dependencies (React Flow, Recharts, react-markdown, Zustand)
6. Create basic app structure (layout, pages)
7. Write basic component tests
8. Run tests and verify they pass
